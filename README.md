# Running Hub

An opinionated, **scored** database of running shoes (then gear, then races). Every entry gets a
transparent, auto-computed score from public structured data — the methodology *is* the product.

Affiliate-monetised, automated-data, products-first. Phase 1 is **shoes only**.

## Why this exists

The market is full of biased "Top 10" listicles. Our wedge is a **transparent algorithmic score**,
**solo-runner authenticity** (real tested mileage), and **hub breadth** (shoes → gear → races) — not
"another review blog."

## The model in one line

A scoring engine + auto-generated comparison pages + affiliate links.

## Phasing (don't boil the ocean)

1. **Shoes** — nail the scoring engine + ~50–100 current shoes. The affiliate money-maker.
2. **Gear** — reuse the same engine for watches, hydration vests (higher ticket).
3. **Races** — a regional race directory: the SEO/community flywheel that makes it a *hub*.

SaaS-grade affiliate value comes from recurring-friendly running retailers and brand programs
